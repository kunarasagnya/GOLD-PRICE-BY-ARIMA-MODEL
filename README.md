# 💰 Gold Price Forecasting using ARIMA Model

## 📌 Project Overview
This project focuses on forecasting gold prices using Time Series Analysis and the ARIMA (AutoRegressive Integrated Moving Average) model. The goal is to analyze historical gold price data and predict future trends.

---

## 🎯 Objectives
- Analyze historical gold price data
- Apply ARIMA model for forecasting
- Evaluate model performance
- Visualize trends and predictions

---

## 🛠️ Technologies Used
- Python / R
- Time Series Analysis
- ARIMA Model
- Pandas, NumPy, Matplotlib

---

## 📊 Methodology
1. Data Collection
2. Data Preprocessing
3. Stationarity Check
4. Model Selection (ARIMA)
5. Model Training
6. Forecasting
7. Evaluation

---

## 📈 Results
- Successfully forecasted gold prices using ARIMA
- Achieved reliable trend predictions
- Visualized actual vs predicted values
